# 🧊 Fridge Object Detection App

Eine einfache Streamlit-App zur Objekterkennung in Kühlschrankbildern mit YOLOv8.

---

## 🚀 Features

- Upload von Bildern
- Objekterkennung mit YOLOv8 (vortrainiert)
- Anzeige von Bounding Boxes
- Liste erkannter Objekte mit Confidence-Werten
- Einfache und übersichtliche UI
